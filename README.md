# Expense Tracker - Bootstrap Blue Theme

This is a standalone expense tracker built with HTML, CSS, JavaScript, and Bootstrap 5.

## Files

- `index.html` - Main page markup with Bootstrap layout, cards, forms, and modal.
- `style.css` - Custom blue theme based on the provided palette.
- `script.js` - Transaction data, INR currency formatting, filters, totals, chart rendering, and add transaction logic.

## Features

- Indian rupee currency formatting.
- Balance, total income, and total expenses summary.
- Spending by category donut chart.
- Transaction filters by category and type.
- Add transaction modal form.
- Responsive desktop and mobile layout.

## How to Run

Open `index.html` in a browser.

For best results, keep an internet connection active because Bootstrap is loaded from the official CDN.
